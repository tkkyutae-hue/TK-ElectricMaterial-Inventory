INLINE ADD ROW (on "Add Equipment" click)
──────────────────────────────────────────
Appends a new <tr class="row-new"> to tbody.
Row background: #f8fdf9 (master cells), #f0fbf5 (live cells)
Border-bottom: 1px solid #d1fae5
Scroll into view smoothly.

Cell inputs in order:

  EQ NO. cell:
    <input> — free text, typeable
    font: 'SF Mono' monospace, 11px bold, uppercase
    placeholder: auto-generated fallback (e.g. "EQ-007")
    User types rental company's actual equipment number here.
    On save: use typed value, fallback to placeholder if empty.

  EQUIPMENT NAME cell:
    Row 1: <select> for equipment type
      Options: Scissor Lift, Boom Lift, One Man Lift,
               Reach Forklift, Forklift, Trench, Excavator
      onChange: auto-populate Size and Brand dropdowns
    Row 2: <input> for S/N — smaller font (10px), monospace placeholder

  SIZE / SPEC cell:
    <select> — options populated based on equipment type selection:
      Scissor Lift:   19 ft, 26 ft, 32 ft, 40 ft
      Boom Lift:      45 ft, 60 ft, 80 ft
      One Man Lift:   12 ft, 20 ft, 26 ft
      Reach Forklift: 3,000 lbs, 5,000 lbs, 8,000 lbs
      Forklift:       3,000 lbs, 5,000 lbs, 8,000 lbs, 15,000 lbs
      Trench:         4 ft, 6 ft, 8 ft, 10 ft
      Excavator:      2 ton, 5 ton, 8 ton, 20 ton
    Before type selected: shows "— Size" placeholder (muted)

  BRAND cell:
    <select> — options populated based on equipment type selection:
      Scissor Lift / Boom Lift / One Man Lift: JLG, Genie, Skyjack, (Hy-Brid/Manitowoc)
      Reach Forklift / Forklift:               Toyota, Caterpillar, Hyster, Crown/Komatsu
      Trench:                                  Vermeer, Ditch Witch, Case
      Excavator:                               Caterpillar, Komatsu, Volvo, Hyundai
    Before type selected: shows "— Brand" placeholder (muted)

  LOCATION cell:
    <input> free text, placeholder "Location"

  ASSIGNED PROJECT cell (live):
    Static "—" text, muted (#c0c4cc) — not editable in add row

  STATUS cell (live):
    Static "—" text, muted — not editable in add row

  LAST UPDATED cell (live):
    Static "Not yet reported" text, muted — not editable in add row

  Actions cell:
    Save button: green #1a472a, white text, 11px bold
    Cancel button: 26×26px gray, hover → red

On Save:
  1. Read EQ NO. input (or use placeholder as fallback), uppercase
  2. Build static row with all entered values
  3. Unassigned project chip + "Not reported" status + "Awaiting report" updated
  4. Flash row green (#dcfce7) for 800ms then fade to normal
  5. Row stays in table as a normal registry row

On Cancel:
  Remove the new row from DOM.
