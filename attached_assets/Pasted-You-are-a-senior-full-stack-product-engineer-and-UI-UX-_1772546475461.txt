You are a senior full-stack product engineer and UI/UX designer.

Build a **production-style inventory management web app** in **Replit** for a **U.S. electrical contractor / electrical materials company**.

This app is for managing real electrical inventory used in commercial and industrial construction projects in the United States.

## Main Objective

Create a **modern, highly intuitive, fast, visually clear inventory management system** that lets users instantly understand:

* what materials are in stock
* what is low
* what is out of stock
* what has been issued to jobsites
* what has been received
* what needs to be reordered
* where every material is stored
* which project consumed which material

The app must feel like a **real internal business tool**, not a beginner demo.

## Development Requirements

Build this project directly in **Replit** with clean architecture and code organization.

### Preferred Tech Stack

* Frontend: React
* Backend: Node.js + Express
* Database: SQLite for initial Replit build
* Styling: Tailwind CSS
* Charts: Recharts
* Icons: Lucide React
* Routing: React Router
* State management: simple and maintainable, prefer React hooks/context unless truly necessary
* Form handling: clean and practical
* Use reusable components throughout
