[VoltStock UI 크기/가독성 조정 요청 — “직관적” 사이즈로 고정]

목표:
- Mode Select(필드/어드민) 화면과 Field Actions 화면의 카드/아이콘/텍스트 크기를 “한눈에 보이는” 수준으로 재조정한다.
- 너무 작지도 크지도 않게: 데스크탑(1440px), 태블릿(1024px), 모바일(390px)에서 자연스럽게 보이게 한다.
- 현재처럼 화면이 비어 보이거나(너무 작음) 카드가 과하게 커 보이는(너무 큼) 문제를 제거한다.

공통 규칙(중요):
1) 카드 터치 영역: 최소 높이 220px~260px(데스크탑), 태블릿 200px, 모바일 160px 이상.
2) 아이콘 크기: 카드 안에서 “대충 1/3 정도” 차지:
   - desktop: 56~64px
   - tablet: 48~56px
   - mobile: 44~52px
3) 제목 폰트:
   - desktop: 18~20px
   - tablet: 16~18px
   - mobile: 16px
4) 설명 폰트:
   - desktop: 13~14px
   - mobile: 12~13px
5) 카드 내부 요소는 모두 중앙 정렬 + 간격 일정 (아이콘-제목-설명 사이 gap 유지)

========================================
A) Select Your Mode 화면 개선
========================================
현재 문제:
- 카드와 아이콘이 너무 작아서 화면이 비어 보임.

수정:
- 화면 중앙에 2개의 카드만 배치하되 카드 자체를 키움.
- 카드 폭:
  - desktop: 각 카드 420~520px 정도
  - tablet: 360~420px
  - mobile: 세로 스택(100%), 카드 1열

구현 힌트(Tailwind 기준 예시):
- 컨테이너: max-w-5xl mx-auto px-6
- 그리드: grid gap-8 md:grid-cols-2
- 카드: min-h-[260px] md:min-h-[280px] rounded-2xl border shadow-sm hover:shadow-md transition
- 아이콘 배경: 연한 배경색(필드=연녹, 어드민=연주황) + 아이콘은 진한 색
  - 아이콘 래퍼: w-16 h-16 (md) / w-14 h-14 (sm)
- 제목: text-lg md:text-xl font-semibold
- 설명: text-sm text-muted-foreground
- “Enter Mode” 같은 문구/버튼은 제거하고 카드 전체 클릭으로 진입

========================================
B) Field Actions 화면 개선
========================================
현재 문제:
- Inventory가 너무 길게 눌려 보이거나, 아래 카드들이 너무 크거나 작게 보임.

원하는 구조:
- 상단에 Inventory 카드 1개를 “가로로 넓게” (가장 중요)
- 하단에 3개 카드: Receive/Return, Issue/Ship, Transactions (같은 크기 3열)
- 모바일에서는 1열로 순서대로 내려오게.

수정:
- 상단 Inventory 카드:
  - desktop: min-h 140~160px, 아이콘 56px, 제목 크게
- 하단 3카드:
  - desktop: min-h 200~240px
  - 아이콘 56px, 제목 18px
- 카드 배경색(흰색만 말고):
  - Inventory: 연한 민트(#EAF7EE) 배경 + 테두리 진녹(#0A6B24)
  - Receive/Return: 연녹 배경
  - Issue/Ship: 연블루 배경(너무 진하지 않게)
  - Transactions: 연그레이/연라일락 배경
  (배경은 “아이콘 뒤 원형 배경 + 카드 상단 아주 연한 틴트”로만 적용하고 카드 전체를 진하게 칠하지 말 것)

구현 힌트:
- 레이아웃: grid gap-6
- 상단 Inventory 카드: col-span-full
- 하단 3카드: grid-cols-1 md:grid-cols-3
- 아이콘 래퍼: rounded-xl w-14 h-14 md:w-16 md:h-16
- 카드 패딩: p-6
- 카드 내부: flex flex-col items-center justify-center gap-3 (하단 3카드)

네비게이션:
- Field Actions 포함 “필드 모드의 모든 페이지” 상단바 왼쪽에 TK 로고 + TK Electric 텍스트 고정.
- 상단바 오른쪽에 Home 아이콘(Mode Select로), Back 버튼(뒤로가기) 둘 다 제공.
- Back은 history.back, Home은 /home(Mode Select)로 이동.

========================================
C) 적용 파일 범위
========================================
- Mode Select 화면 컴포넌트(예: Home/ModeSelect)
- FieldLayout(상단바)
- FieldHome(Field Actions)
- 관련 CSS/Tailwind 클래스 조정

========================================
D) 검증 체크
========================================
- 1440px: 카드가 작아 보이지 않고 화면이 허전하지 않아야 함.
- 1024px: 2열/3열 유지하며 아이콘/텍스트 균형 유지.
- 390px: 카드가 눌려보이지 않고 스크롤로 자연스럽게 내려가며 터치하기 쉬워야 함.
- “아이콘이 너무 작다/크다” 느낌 없이 한눈에 기능이 구분되어야 함.

위 요구사항대로 “직관적인 크기”로 다시 조정해줘.
