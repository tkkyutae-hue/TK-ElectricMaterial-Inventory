=====================================================
Family examples: Tray / Covers / Fittings / Connectors

1) Cross:
   - SKU starts with "CT-CRS-" OR name ILIKE '%cross%'
2) Tee:
   - SKU starts with "CT-TEE-" OR name ILIKE '%tee%'
3) Elbow:
   - SKU contains 'ELB' OR name ILIKE '%elbow%'
4) Reducer:
   - name ILIKE '%reducer%' OR '%reduce%'
5) Vertical:
   - name ILIKE '%vertical%'
6) Horizontal:
   - name ILIKE '%horizontal%'
7) Other

For Cable Tray -> Family=Connectors:
