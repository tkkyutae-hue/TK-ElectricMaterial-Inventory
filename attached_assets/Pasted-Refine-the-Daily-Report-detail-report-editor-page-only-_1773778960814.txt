- Trade / Classification (auto-filled from selected worker, read-only)
- Attendance Status
- Start Time
- End Time
- Work Hours (auto-calculated)
- Notes

- ATTEND
- PTO
- SICK
- ABSENT
- OFF
- LATE
- EARLY LEAVE
- WFH
- TRAINING
- SUSPENDED
- TERMINATED

- Trade / Classification auto-fills after worker select
- Work Hours auto-calculate from Start/End unless attendance status is ABSENT / OFF / PTO etc

- Add Row button remains at bottom
- allow delete row

- Total Man-Hours

This section should feel similar to the Excel manpower sheet logic, but modern and app-friendly.
