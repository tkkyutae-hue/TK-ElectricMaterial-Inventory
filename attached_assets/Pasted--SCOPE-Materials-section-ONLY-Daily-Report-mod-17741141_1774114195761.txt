CHANGE 3 — SIZE extraction from item name (display only)

  Extract size token from the inventory item name and show in SIZE column.
  Remove the extracted token from the MATERIAL NAME display.
  Do NOT modify the underlying data.

  Extraction rules (in priority order):
    1. Starts with #N or #N/N     → e.g. "#1", "#10", "#4/0"
       SIZE: "#1"   NAME: "(3C+G) Cable"
    2. Starts with N" or N/N"     → e.g. '3/4"', '1"', '2"'
       SIZE: '3/4"' NAME: "EMT Conduit"
    3. Starts with N.N kVA / NkVA → e.g. "112.5 kVA"
       SIZE: "112.5 kVA"  NAME: "480V-208Y/120V Transformer..."
    4. Starts with NA / NmA       → e.g. "100A", "200A"
       SIZE: "100A"  NAME: "240V / NEMA1 Fusible Disconnect Switch"
    5. No match → SIZE shows "—" (muted), NAME unchanged
