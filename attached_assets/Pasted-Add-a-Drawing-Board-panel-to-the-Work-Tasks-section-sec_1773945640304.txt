════════════════════════════════════════
PIN SYSTEM
════════════════════════════════════════
Pins are circular markers placed on the drawing.
Each pin has an ID: A1, A2, A3... (auto-incremented)

PIN STATES:
  Linked pin   → green circle (#16a34a), white text
  Unlinked pin → blue circle (#1d6ecc), white text

PIN SIZE: 26×26px, border-radius: 50%
PIN SHADOW: box-shadow: 0 2px 8px rgba(0,0,0,0.25)

ADD PIN MODE:
  When user clicks [📍 Add Pin] button:
    - Button becomes active/highlighted (purple style)
    - Cursor changes to crosshair on the canvas
    - Clicking the canvas places a new pin at that position
    - New pin gets next auto ID (A1, A2, ...)
    - New pin is "unlinked" (blue) by default
    - Pin placement stores: x%, y% (percentage of canvas size)
      so it stays in position if canvas resizes
    - After placing, Add Pin mode stays active
      user can keep placing more pins
    - Clicking [📍 Add Pin] again or pressing Escape deactivates

CLICKING AN EXISTING PIN:
  Show a small popover/tooltip above the pin:
    Pin ID: "A1"
    Linked task: "Install EMT Conduit" (or "Not linked")
    Actions:
      [Link to task ▾]  [Rename]  [Delete]
  "Link to task" opens a small dropdown of existing task descriptions
  Selecting a task links this pin to that task

TECHNICAL IMPLEMENTATION NOTES
════════════════════════════════════════
Pin positions:
  Store as percentage values (x: 42.5%, y: 30.2%)
  not pixel values — so layout resizing doesn't break positions

Pin data structure per pin:
  id: "A1"
  x: 42.5   (% from left)
  y: 30.2   (% from top)
  linked_task_index: null or number (index of task in list)

Drawing image:
  Store as a data URL or blob URL in component state
  No backend upload needed for now — local state only
  (can be wired to real storage in a later step)

When a task is deleted:
  Any pin linked to that task becomes unlinked (blue)
  Pin stays on the drawing

When a pin is deleted:
  The linked task's Drawing Ref badge resets to "📍 Link pin"
