════════════════════════════════════════
CORE PROBLEM TO FIX
════════════════════════════════════════
The current category system mixes conduit runs and fittings/connectors
into the same or adjacent categories (e.g. "Flexible" contains both
the conduit itself and its connectors).

This becomes a problem when EMT and Rigid conduit are added later —
connectors and runs will be impossible to distinguish at a glance.

The fix: separate "Conduit" (runs only) from "Fittings & Connectors"
(all connectors, couplings, straps, bushings) into distinct top-level categories.

════════════════════════════════════════
NEW CATEGORY SYSTEM — 8 top-level groups
════════════════════════════════════════
Replace the current category grouping with this fixed 8-category system.
Display them in this exact order, top to bottom:

  1. Conduit
  2. Fittings & Connectors
  3. Cable Tray
  4. Cable / Wire
  5. Grounding
  6. Boxes
  7. Devices
  8. Equipment

════════════════════════════════════════
CATEGORY DEFINITIONS
════════════════════════════════════════
Use these rules to assign existing scope items to the new categories:

1. CONDUIT
   → Conduit runs (the pipe/tube itself) only
   → Includes: Liquidtight Flexible Conduit, EMT Conduit, Rigid Conduit, MC Cable (if applicable)
   → Does NOT include connectors, couplings, or fittings

2. FITTINGS & CONNECTORS
   → All connector/coupling/strap/bushing/elbow items regardless of conduit type
   → Includes: Liquidtight Connector Straight, EMT Set Screw Connector,
     EMT Coupling, EMT Strap, Conduit Clamp, Ground Bushing, Plastic Bushing,
     Unistrut Pipe Clamp, and similar fitting items
   → This category catches fittings for ALL conduit types in one place

3. CABLE TRAY
   → Cable tray straight sections, covers, and tray fittings
   → Includes: Cable Tray Straight, Checkered Cable Tray Cover,
     Cable Tray Horizontal Elbow, Cable Tray Vertical Out

4. CABLE / WIRE
   → Wire and cable runs
   → Includes: Multi-Conductor cable, Single Conductor wire, THHN, etc.
   → Does NOT include ground wire (see Grounding)

5. GROUNDING
   → Grounding wire and bonding items
   → Includes: #2/#6/#10 Grounding Wire, ground bars, ground rods

6. BOXES
   → Enclosures and boxes
   → Includes: Junction Box, Pull Box, Metal Square Deep Box, Panel

7. DEVICES
   → Electrical devices installed at endpoints
   → Includes: Receptacle, Switch, Outlet, Lighting Fixture, Exit Sign

8. EQUIPMENT
   → Major electrical equipment
   → Includes: Transformer, Motor, MCC, Panel Board, Special Equipment
